Devices featured on your browser

In addition to the help that is directly available here, this website has also been designed so that you can take full advantage of your browser's features.

Enlarging the font size

You have the possibility to enlarge the font size on all the website's pages.

  • On a PC: Press Crtl + + to enlarge the font size. Pressing Crtl + - enables to reduce the font size. Successive presses of these keys allow you to enlarge or reduce the font size in steps. Also note that pressing Ctrl + 0 (the number "zero") allows you to return to the default text size.
  • On a Mac: In the shortcuts above, just replace the Ctrl key with the Cmd key to perform the same actions.

Keyboard navigation

This website has been designed to allow navigation on all of its pages using the keyboard. In most cases, you just have to use the Tab key on your keyboard to successively navigate from one clickable element to another clickable element on the page. Conversely, if you want to navigate back one step, use the Shift + Tab shortcut.

Printing

You also have the possibility to print any page on the website by accessing the desired page then pressing the Ctrl + P keys simultaneously on a PC or Cmd + P on a Mac.

About us

About amendes.gouv.fr

The www.amendes.gouv.fr website was subject to a declaration to the CNIL (Commission nationale de l'informatique et des libertés), the French Data Protection Authority, which resulted in an order of 24 January 2007, amending the decree of 13 January 2004, creating the e-payment system of fines (Official Gazette of 22 February 2007).
